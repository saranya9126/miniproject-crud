# Restaurant CRUD Backend

A RESTful API for managing restaurants using Node.js, Express, and MongoDB.

## Features
- Create restaurant
- Get all restaurants
- Get restaurant by ID
- Update restaurant
- Delete restaurant

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose

## API Endpoints
POST    /api/restaurants  
GET     /api/restaurants  
GET     /api/restaurants/:id  
PUT     /api/restaurants/:id  
DELETE  /api/restaurants/:id  

## Run Locally
npm install  
npm run dev
